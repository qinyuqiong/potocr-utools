# uTools POT OCR/翻译插件

通过 HTTP 外部调用 [POT 桌面应用](https://github.com/pot-app/pot-desktop) 实现 OCR 和翻译功能的 uTools 插件。

## 功能

- 📸 **截图OCR** - 调用 POT 截图，识别图片中的文字
- 🌍 **截图翻译** - 调用 POT 截图，直接翻译图片中的文字
- ✏️ **输入翻译** - 输入文本进行翻译
- ⚙️ **可配置** - 支持配置 POT API 地址（默认 `http://127.0.0.1:60828`）
- 📎 **支持粘贴图片** - 将图片粘贴到 uTools 可直接处理

## 使用前提

1. 安装 [POT 桌面应用](https://github.com/pot-app/pot-desktop)
2. 在 POT 设置中开启「外部调用」，记住端口号（默认 60828）
3. 如果端口修改了，请在本插件的「pot设置」中更新配置

## 安装

### 开发模式

1. 克隆或下载本项目
2. 安装依赖：

```bash
npm install
```

3. 构建：

```bash
npm run build
```

4. 在 uTools 开发者工具中，导入 `dist` 目录

### 使用已构建版本

直接将 `dist` 目录压缩为 zip，在 uTools 开发者工具中导入

## 触发方式

| 功能 | 触发关键字 | 自动匹配 |
|------|-----------|----------|
| 截图OCR | `截图OCR`, `potocr`, `ocr识别` | 图片（粘贴/拖入） |
| 截图翻译 | `截图翻译`, `pot翻译` | 图片（粘贴/拖入） |
| 输入翻译 | `输入翻译`, `pot翻译` | 文本（输入） |
| 设置 | `pot设置`, `pot配置` | - |

## 工作流程

### 截图OCR / 截图翻译

1. 在 uTools 输入关键字触发，或者粘贴/拖入图片
2. 如果是关键字触发，插件直接调用 POT，**POT 会自己弹出截图**
3. POT 完成识别/翻译后，会自己弹出窗口显示结果
4. 插件自动隐藏 uTools 窗口

### 输入翻译

1. 在 uTools 输入文字触发，或者直接输入文本后点击翻译
2. 插件发送文本到 POT，POT 弹出窗口显示翻译结果
3. 插件自动隐藏 uTools 窗口
